## Getting Started

### Prerequisites

- Node.js (LTS version)
- npm or yarn
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/vernier-caliper-simulator.git

# Navigate to project directory
cd vernier-caliper-simulator

# Install dependencies
npm install

# Start development server
npm run dev
```
